# Truck Managment Logistics
---
Este proyecto fue desarrollado con el objetivo de generar una aplicación de escritorio que permita al usuario final administrar aspectos relacionados al negocio de administración de viajes y camiones, de forma que pueda tener un espacio donde registrar detalles propios del negocio.

---
# Tecnologías utilizadas
- C#
- Entity Framework Core
- MySQL
- WinForms
